# 로봇이 멈췄을 때 대처법

> 기준 매뉴얼: RB Series 사용자 매뉴얼 V6.3 (8.3장 / 9.5장)

## 빠른 판단 트리

    로봇이 멈췄다
    ├── 팝업 메시지가 뜨는가?
    │   ├── "외부 충돌 감지"  → 1. 외부 충돌 참고
    │   ├── "자가 충돌"       → 2. 자가 충돌 참고
    │   └── "알람 메시지"     → Resume 또는 Halt 선택
    └── 팝업 없이 멈춤
        ├── PC/태블릿 분리됨   → 케이블 확인
        ├── 비상정지 눌림      → 스위치 해제 후 재초기화
        └── 관절 제어기 에러   → 4. 비상 관절 복구 참고

## 1. 외부 충돌 (Out Collision)

**증상:** 충돌 감지 팝업 출현, 동작 일시정지
**충돌 코드:** `OUT`(과전류), `ACC` / `GYR`(순간 진동)

**대처:**

1. 충돌 요인 제거 (장애물 치우기)
2. `[계속(Resume)]` — 작업 재개 / `[정지(Halt)]` — 작업 종료

**TOK TOK 회복법:** 로봇 말단을 가볍게 **2회 두드리면** Resume과 동일 효과

**오감지가 반복된다면:**

- 툴 부하(Load) 세팅이 정확한지 확인 (Setup → Tool)
- 로봇 중력 방향 설정 확인

## 2. 자가 충돌 (Self Collision)

**증상:** 작업 범위 이탈 또는 자체 충돌 위험 자세 근접 시 정지

**대처:**

1. 티칭 버튼을 눌러 직접교시로 로봇을 정상 자세로 이동
2. 티칭한 모션 경로 및 작업 범위 설정 재검토

**시뮬레이션 모드에서 발생 시:**

- 모션 관련 버튼으로 회복 시도
- Real 모드로 잠시 전환 후 실제 관절 정보로 동기화
- 티칭 버튼으로 직접 회복

## 3. 기타 정지 상황

| 상황 | 원인 | 대처 |
|------|------|------|
| 알람 메시지 | 프로그램 Alarm 기능 실행 | Resume 재개 / Halt 종료 |
| PC/태블릿 분리 | 케이블 손상, 연결 끊김 | USB/LAN 케이블 교체, 다른 포트 시도 |
| 외부 전력 중단 | 비상정지 스위치 눌림 | 스위치 복구 후 재초기화 |
| 관절 제어기 에러 | BIG/JAM/CUR/Temperature/Mode | 4. 비상 관절 복구 참고 |

**관절 에러 코드:**

- `BIG` : 지령-엔코더 차이 과다
- `JAM` : 전류는 흐르나 엔코더 무변화
- `CUR` : 과전류 지속
- `Temperature` : 보드 과열
- `Mode` : 제어모드 불일치

## 4. 비상 관절 복구 (Emergency Joint Recovery)

> ⚠️ **브레이크 해제 시 팔이 중력으로 떨어질 수 있습니다. 반드시 로봇 팔을 손으로 지지하세요.**

**경로:** Setup → Utility → 관절 유지 및 보수

1. 복구할 관절 선택 (초록=정상 / 빨강=확인 필요)

       0=Base / 1=Shoulder / 2=Elbow / 3=Wrist1 / 4=Wrist2 / 5=Wrist3

2. `[풀기(Release)]` 클릭 — 로봇 팔 반드시 손으로 지지
3. 관절의 홈(정렬 표시) 정렬
4. `[초기화]` 클릭
