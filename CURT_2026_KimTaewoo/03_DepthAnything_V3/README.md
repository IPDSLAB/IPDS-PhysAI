# Depth Anything V3

## Objective

Jetson Orin NX 환경에서 Depth Anything V3를 실행하고 실시간 추론 환경을 구축한다.

## Installation

Depth Anything V3 및 관련 라이브러리를 설치하였다.

## Webcam Integration

Logitech C922 웹캠을 연결하여 실시간 영상을 입력받았다.

## Optimization

실시간 추론 속도 향상을 위해 입력 해상도 및 추론 주기를 조정하였다.

## Result

실시간 Depth Estimation 데모를 성공적으로 수행하였다.
