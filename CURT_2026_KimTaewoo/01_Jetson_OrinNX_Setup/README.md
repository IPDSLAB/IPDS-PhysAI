# Jetson Orin NX Setup

## Objective

Jetson Orin NX에서 AI 비전 모델 실행을 위한 개발 환경을 구축한다.

## Environment

- Jetson Orin NX 16GB
- Ubuntu 22.04
- JetPack 6.x
- CUDA 12.x

## Procedure

### 1. SDK Manager 설치

Windows 노트북에서 SDK Manager를 사용하여 Jetson Orin NX에 Ubuntu 및 JetPack을 설치하였다.

### 2. Ubuntu 초기 설정

사용자 계정 생성 및 네트워크 설정을 진행하였다.

### 3. CUDA 확인

CUDA 정상 동작 여부를 확인하였다.

### 4. Python 환경 구성

가상환경 생성 후 필요한 라이브러리를 설치하였다.

## Result

Jetson Orin NX 개발 환경 구축 완료.
