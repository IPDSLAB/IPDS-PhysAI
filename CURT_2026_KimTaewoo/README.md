# CURT 2026 - Jetson Orin NX & Depth Anything

## Overview

본 프로젝트에서는 Jetson Orin NX 개발 보드를 활용하여 Ubuntu 개발 환경을 구축하고, 단안 깊이 추정 모델인 Depth Anything V2 및 V3를 설치 및 실행하였다.

또한 실시간 웹캠 기반 추론 환경을 구성하여 Jetson 환경에서의 동작 가능성을 검증하였다.
